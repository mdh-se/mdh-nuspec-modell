# mdh-nuspec-modell

En modell för att hantera nuspec-filer i Java. Dessa filer har filändelsen .nuspec. Detta projekt innehåller versionen 2010/07 av en xsd från [NuGets GitHub repository](https://github.com/NuGet/NuGet.Client/blob/dev/src/NuGet.Core/NuGet.Packaging/compiler/resources/nuspec.xsd).

Dokumentationen för .nuspec-filer finns [här](https://docs.microsoft.com/en-us/nuget/reference/nuspec).
